Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer.
    Karena tag article memiliki type block, artinya elemnt ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misalnya kita memiliki halaman blog, maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis.
    Karena aside memiliki type/sifat block mka lebarnya sesuai page jika kita tidak menentukan lebarnya.
